# Stitch v1 — archived 2026-05-16

**Do not use these assets.** They are kept for reference, not for production.

## What this is

Two homepage mockup attempts (`homepage-v1.png`, `homepage-v2.png`) and four
brand assets (`ship.png`, `ship-minimal.png`, `helmsmen.png`, `shield-base.png`,
`wordmark.png`) generated by Google Stitch (`stitch.googleapis.com/mcp`)
during the Track B rebrand prototyping on 2026-05-16.

## Why they're archived

Stitch was not the right tool for the visual rebrand:

- **v1** came back overdesigned — ornate gold scrollwork, heraldic frames,
  woodcut texture, Latin-style decoration, helmsmen portraits, and a
  "tavern-sign" tall-ship aesthetic. The brand needs a modern AI agency
  feel, not Renaissance maritime art.
- **v2** stripped most of that on a hard reset and introduced purple as
  the brand secondary, but the ship asset collapsed to a flag-with-an-
  underline silhouette and the overall layout still read as generic.

The conclusion: Stitch's UI-design output is fine for utility screens
but doesn't have the craft ceiling this rebrand needs. The visual
direction will be sourced externally (Figma / Dribbble references,
hand-built mocks, or a different image generator) before any further
work on `rebrand/phase-2`.

## What survives in the active codebase

Nothing in the live or rebrand code references these PNGs. The deps,
tokens, and route scaffolding that landed on `rebrand/phase-2` during
Track B prototyping (`framer-motion`, `react-intersection-observer`,
`navy` + `gold` Tailwind tokens, Cormorant font load, `AppShell` route-
aware chrome, `/v2` route) are general-purpose and stay in place. The
nautical placeholder SVG components that *did* depend on this aesthetic
were removed in the same commit that produced this archive.
