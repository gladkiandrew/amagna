05Enhancement toolkit — closing the gap to "barely AI"

  

    
    
      
  • Image-first — generate/approve a perfect still, then animate it. Biggest single quality lever; fixes most teeth/identity/geometry breaks.
    • 
      
  • Upscale — render 720/1080p, then AI-upscale toward 4K. Removes softness, cheaper than rendering huge.
    • 
      
  • Post-production = 50% of the wow — color grade (one house LUT), music (carries emotion), burned captions, sound design, cut the best 2–3s.
    • 
      
  • Volume → cherry-pick — ~1 in 3–5 generations is usable. Generate several, keep the best beats.
    • 
      
  • Stitch short beats — 2–3 clean short clips edited together beat one risky 15s shot.
    • 
      
  • Reference / Soul consistency — reuse a reference image or trained Soul so the same look recurs across a campaign.
    • 
      
  • Compositing tools — remove-background (product on a clean branded backdrop), outpaint (extend to 9:16), motion-control (directed camera), upscale-image (sharpen a hero still before animating).
    • 
      
  • House look — one grade + caption style across all Amagna content so it reads like an agency, not a hobby.
