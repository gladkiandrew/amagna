# Example videos (homepage "See the output" swipe deck)

Drop the real MP4s here with these exact names and the cards become live `<video>`
players automatically (muted, looped, inline). Until a file exists, the card
shows a styled "Example drop incoming" placeholder.

- `example-1.mp4`
- `example-2.mp4`
- `example-3.mp4`

Format: 9:16 (vertical / TikTok), H.264 MP4, muted. Keep them short and light
(a few MB each) — they autoplay on the homepage.
